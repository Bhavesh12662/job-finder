# 🚀 Job Finder App

A modern IT job dashboard that helps developers find real jobs and internships across India and remote.


## 🔥 Features

* ✅ Real-time jobs from API (no scraping)
* 🎯 Filter by categories (Frontend, Backend, Full Stack, AI/ML, DevOps)
* 💾 Save jobs (stored in localStorage)
* ✅ Mark jobs as applied
* 🔍 Search functionality
* 📱 Fully responsive UI
* 🎨 Premium dark theme design



## 🛠 Tech Stack

* HTML
* Tailwind CSS
* JavaScript (Vanilla)
* Node.js (Express backend)
* External Job API (Remotive)
